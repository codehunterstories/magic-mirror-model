# MagicMirror Modules

MagicMirror 是一个非常受欢迎的开源项目，允许用户创建自己的智能镜子，可显示各种有用的信息，例如天气、新闻摘要和日历事件等。此仓库主要是基于树莓派，编写魔镜相关的模块。
